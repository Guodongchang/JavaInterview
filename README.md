## 学习方法

- 稍微简单的知识，自己通过书籍，进行学习，并自己写代码实际操作总结。
- 稍微复杂一点的知识，可以先看看学习视频（比如B站一些学习视频，然后自己深入学习可以看书或者看文档），并进行总结。
- 每天选择一两篇写的好的博客，进行认真学习，总结。不管关注的公众号多少，或者看的博客多少，只有自己仔细研读过的文章，并进行总结，才可能消化里面的知识。
